# Cato 🐱

A friendly, **bilingual (English + 中文)** reminder & habit app you talk to like a cute cat.
Built with Expo + React Native + TypeScript, targeting iOS. Made as a gift.

> Type a sentence ("remind me to call mom tomorrow at 5pm" / "明天早上七点去签证处") and the
> cat creates the reminder, nudges you at the right time, and tracks your habit streaks.

## Status
Early development on branch `feature/cato-app`. Building in documented, beginner-friendly
steps.
